Apa itu database?
database adalah sistem untuk menyimpan, mengelola, dan mengakses data secara efisien 

Peran dalam website :
- menyimpan data pengguna, transaksi, produk, dll
- membantu pengelolaan konten dinamis
- mendukung analisis dan pelaporan data
Contoh :
- Media sosial : menyimpan profil pengguna dan postingan
- e commerce : mengelola pembelian, produk,transaksi, dll


Jenis - jenis database :
1. Relasional :
- MySql, PostgreSql
- data disimpan dalam tabel yang berelasi
- menggunakan SQL (Structure Query Language)

2. Non Relasional :
- MongoDB, Firebase
- Data disimpan dalam bentu dokumen, key-value, atau graf

Kapan digunakan?
- SQL jika untuk data terstruktur
- Non SQL untuk data tidak terstruktur, membutuhkan skalabilitas tinggi
