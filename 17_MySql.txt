Apa itu database?
database adalah sistem untuk menyimpan, mengelola, dan mengakses data secara efisien 

Peran dalam website :
- menyimpan data pengguna, transaksi, produk, dll
- membantu pengelolaan konten dinamis
- mendukung analisis dan pelaporan data
Contoh :
- Media sosial : menyimpan profil pengguna dan postingan
- e commerce : mengelola pembelian, produk,transaksi, dll


Jenis - jenis database :
1. Relasional :
- MySql, PostgreSql
- data disimpan dalam tabel yang berelasi
- menggunakan SQL (Structure Query Language)

2. Non Relasional :
- MongoDB, Firebase
- Data disimpan dalam bentu dokumen, key-value, atau graf

Kapan digunakan?
- SQL jika untuk data terstruktur
- Non SQL untuk data tidak terstruktur, membutuhkan skalabilitas tinggi

FOREIGN KEY
tali penghubung antar tabel untuk identifikasi antar Data
- Primary key : Unik, tidak null digunakan untuk identifikasi setiap baris
- FOREIGN key : menghubungkan setiap tabel untuk identifikasi antar data
- Unique : nilai kolom harus unik (tidak ada duplikat)
- Not Null : nilai kolom tidak boleh kosong
- Default : memberikan nilai default jika tidak ada input
- Check : membatasi nilai data dengan syarat tertentu
- Index : mencari data dengan cepat (berlaku untuk database besar)
- Auto Increment : menghasilkan nilai unik / ciri khas data secara otomatis
note : nilai unik adalah cirikhas suatu data yang tidak dimiliki oleh data lainya, sehingga semua data memiliki ciri khas nya masing masing
contoh : [[Prabowo, 2], [Gibran, 1]]
data di atas menunjukan nilai unik Prabowo adalah 2 dan Gibran 1 


Struktur dasar database MySql
- database : tenpat utama menyimpan data
- tabel : struktur data berisi kolom dan baris
- primary key : kolom unik sebagai indentitas data
- relasi tabel : hubungan antar tabel


Perintah dasar MySql :
CREATE DATABASE bootcamp
USE bootcamp

membuat tabel :
    CREATE TABLE users (
        id INT AUTO_INCREMENT PRIMARY KEY,
        name VARCHAR(100),
        email VARCHAR(100)
    );

menambahkan data :
    INSERT INTO users (name, email) VALUES ('vero', 'royyankebonharjo@gmail.com');

mengambil data :
    SELECT * FROM users;

memperbarui data :
    UPDATE users SET name = 'vero' WHERE id = 1;

menghapus data :
    DELETE FROM users WHERE id = 1;
